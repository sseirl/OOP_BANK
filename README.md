# Personal Account Management

## Описание
Проект реализует управление персональным банковским счетом на Python с использованием объектно-ориентированного программирования (OOP).  
Программа позволяет:
- вести депозит и снятие средств;
- отслеживать баланс;
- сохранять историю транзакций.

## Классы

### Amount
Класс для представления одной транзакции.
- **Атрибуты**:
  - `amount` (float) — сумма транзакции.
  - `timestamp` (datetime) — дата и время транзакции.
  - `transaction_type` (str) — тип транзакции (`DEPOSIT` или `WITHDRAWAL`).
- **Методы**:
  - `__init__` — инициализация транзакции.
  - `__str__` — строковое представление транзакции.

### PersonalAccount
Класс для управления персональным банковским счетом.
- **Атрибуты**:
  - `account_number` (int) — номер счета.
  - `account_holder` (str) — владелец счета.
  - `balance` (float) — текущий баланс.
  - `transactions` (list) — список транзакций (`Amount`).
- **Методы**:
  - `__init__` — инициализация счета.
  - `deposit(amount)` — пополнение счета.
  - `withdraw(amount)` — снятие со счета (проверка на достаточность средств).
  - `print_transaction_history()` — вывод истории транзакций.
  - `get_balance()` — получение текущего баланса.
  - `get_account_number() / set_account_number(account_number)` — доступ и изменение номера счета.
  - `get_account_holder() / set_account_holder(account_holder)` — доступ и изменение владельца счета.
  - `__str__` — строковое представление счета.
  - `__add__(amount)` — эквивалент метода `deposit`.
  - `__sub__(amount)` — эквивалент метода `withdraw`.
